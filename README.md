# Генератор рапортов для ОСБ
Автоматическая генерация BB кода с GUI.

## Как пользоваться
 - Запускаете файл app.py (в репозитории уже лежит .venv, так что должно работать без установки python)
 - Настраиваете ФИО и должность, ФИО и подпись, нажимаете "сохранить".
 - Заполняете рапорт по порядку, в конце можно добавить несколько ссылок на доказательства.
 - Нажимаете "Сгенерировать BB-код" и затем "Скопировать".
Вы получаете готовый BB-код для вставки на форум.
